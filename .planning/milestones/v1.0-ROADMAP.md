# Milestone v1.0: "You Can't Read This" — MVP

**Status:** ✅ SHIPPED 2026-04-19
**Phases:** 1–2
**Total Plans:** 7

## Overview

Two-phase build for the "You Can't Read This" Slidev presentation. Phase 1 delivered everything needed for a live Human Drum session: all 13 slides with content, animations, presenter tools, the backing track player, and a working GitHub Pages deployment. Phase 2 layered in VexFlow drum notation via DrumStaff.vue, elevating the visual teaching slides from color grids to proper percussion staves. PlayableDrumStaff.vue (Tone.js audio + beat cursor) was cut to v2.

## Milestone Summary

**Key Decisions:**
- Slidev over PowerPoint — Vue component support, GitHub Pages deploy
- Deploy base path first within Phase 1 — highest-risk integration, validate early
- VexFlow percussion spike before DrumStaff.vue API — v5 x-notehead API undocumented; spike confirmed `'a/5/x'` syntax
- Hi-hat beams in groups of 2 — standard drum notation convention
- Kick rests hidden via `fillStyle:'none'` — reduces visual clutter without breaking voice tick count
- Removed global white ctx style — seriph theme is light background, not dark
- PlayableDrumStaff.vue deferred to v2 — time constraint, static notation delivers core value

**Issues Resolved:**
- White-on-white stave bug (global ctx white style conflicting with light seriph theme)
- VexFlow v5 percussion API uncertainty (resolved via throwaway spike)

**Issues Deferred:**
- PlayableDrumStaff.vue (Tone.js audio + beat cursor) — v2 scope
- GitHub Actions auto-deploy — v2 scope

**Technical Debt Incurred:**
- None significant — spike-first approach eliminated main risk
