VexFlow named ESM imports:
```typescript
import { Renderer, Stave, StaveNote, Voice, Formatter, Stem, Beam } from 'vexflow'
```

Renderer init (SVG backend):
```typescript
const renderer = new Renderer(containerElement, Renderer.Backends.SVG)
renderer.resize(width, height)
const ctx = renderer.getContext()
ctx.setFillStyle('white')
ctx.setStrokeStyle('white')
```

Percussion stave:
```typescript
const stave = new Stave(10, 30, staveWidth)
stave.addClef('percussion').addTimeSignature('4/4')
stave.setContext(ctx).draw()
```

X-notehead per-key syntax (CRITICAL — not noteType):
```typescript
// single x-notehead:
new StaveNote({ keys: ['a/5/x'], duration: '8', stem_direction: Stem.UP })
// mixed chord (hi-hat x + snare oval):
new StaveNote({ keys: ['a/5/x', 'c/5'], duration: '8', stem_direction: Stem.UP })
```

Color before draw:
```typescript
note.setStyle({ fillStyle: '#eab308', strokeStyle: '#eab308' })          // whole note
chord.setKeyStyle(0, { fillStyle: '#eab308', strokeStyle: '#eab308' })   // per key
chord.setKeyStyle(1, { fillStyle: '#3b82f6', strokeStyle: '#3b82f6' })
```

Two-voice format + draw:
```typescript
new Formatter().joinVoices([v1, v2]).format([v1, v2], staveWidth - 60)
v1.draw(ctx, stave)
v2.draw(ctx, stave)
```
