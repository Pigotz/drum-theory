Install gh-pages:

```bash
npm install -D gh-pages@6.3.0
```

Then update the `scripts` section of `package.json` to contain exactly these three
scripts (add or replace — do NOT remove other scripts the scaffold added):

```json
"scripts": {
  "dev": "slidev --open",
  "build": "slidev build --base /drum-theory/",
  "deploy": "npm run build && touch dist/.nojekyll && npx gh-pages -d dist"
}
```

Key points:
- `build` uses `--base /drum-theory/` to match the headmatter `base:` value — both must
  match exactly or assets 404 on GitHub Pages (Pitfall 1 in RESEARCH.md)
- `deploy` chains: build → create `.nojekyll` in dist → push dist to gh-pages branch
- `touch dist/.nojekyll` prevents GitHub Pages Jekyll from mangling Vite's underscored
  output files (e.g., `_assets/`)
- The scaffold may have added a `build` script already — replace it if so

Do NOT add a `export` script or any GitHub Actions workflow — DEPLOY-03 is v2 scope.
